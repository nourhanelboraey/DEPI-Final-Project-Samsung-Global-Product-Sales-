# DEPI-Final-Project-Samsung-Global-Product-Sales-
The Samsung Global Product Sales Dataset on Kaggle contains structured data about the worldwide sales performance of Samsung products across different regions and time periods. It includes details such as product categories, sales volume, and revenue, making it suitable for analyzing trends and market performance.
## Dataset Source 
https://www.kaggle.com/datasets/ashyou09/samsung-global-product-sales-dataset
